FIVE STEPS

load it in under a minute

this is the standard way browsers let you run an extension straight from a folder, the same flow developers use every day

01

download and unzip

grab the ZIP above, then unzip it. keep the folder somewhere safe, the extension runs straight from it so do not delete it later

02

open the extensions page

in your browser type chrome://extensions in the address bar and press enter. or open the menu, then Extensions, then Manage extensions

03

turn on Developer mode

flip the Developer mode switch in the top right corner of the extensions page. this is what lets a browser run an extension from a folder

04

click Load unpacked

three buttons appear. click Load unpacked, then pick the folder you unzipped, the one that has manifest.json inside

05

pin LENS and go

LENS shows up in your list. click the puzzle icon in the toolbar and pin it so it is one click away. open any X profile and the panels load on their own
